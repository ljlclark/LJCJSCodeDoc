Method Clear

Syntax

JavaScript
Clear()

Clears the collection list.

Example

JavaScript
// Clears the collection list.
Clear()
{
  const testItems = new TestItems();
  let testItem = testItems.Add("NameOne");
  testItem.Sequence = 1;
  testItems.Clear();

  let result = testItems.Count;
  
  // result:
  // 0
}
