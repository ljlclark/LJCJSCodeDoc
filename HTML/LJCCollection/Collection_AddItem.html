Method _AddItem

Parameters
item - The item to add to the list.

Returns

The added item.

Syntax

JavaScript
_AddItem(item)

Adds the item to the array.

Example

JavaScript
// Adds the item to the array.
_AddItem()
{
  const testItems = new TestItems();
  let testItem = new TestItem("NameOne");
  testItem.Sequence = 1;
  testItems._AddItem(testItem);

  let result = testItems.Count;

  // result:
  // 1
}
