Method ShowProperties

Parameters
location - The debug location text.
item - A collection item.

Syntax

JavaScript
static ShowProperties(location, item)

Show the properties of an object that are not null or "" and do not start with "on".

Example

JavaScript
// Show the properties of an object that are not null or "" and do not start
// with "on".
ShowProperties()
{

}
