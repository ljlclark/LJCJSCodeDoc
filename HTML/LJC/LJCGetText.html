Method GetText

Parameters
elementID - The element ID name.

Returns

The element content (inner) text.

Syntax

JavaScript
static GetText(elementID)

Gets the element text.

Example

JavaScript
<body>
  <div id="testDiv"></div>
</body>

// Gets the element text.
GetText()
{
  testDiv.innerText = "X";
  let result = LJC.GetText("testDiv");
        
  // result:
  // "X";

  testDiv.innerText = "";
  result = LJC.GetText("testDiv");
        
  // result:
  // "";
}
