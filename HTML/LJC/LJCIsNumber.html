Method IsNumber

Parameters
text - A string value.

Returns

true if a number; otherwise false.

Syntax

JavaScript
static IsNumber(keyDownEvent)

Checks if the text is a number.

Example

JavaScript
// Checks if the text is a number.
IsNumber()
{
  let number = 1;
  let result = LJC.IsNumber(number);

  // result
  // true

  number = "X";
  result = LJC.IsNumber(number);

  // result
  // false
}
