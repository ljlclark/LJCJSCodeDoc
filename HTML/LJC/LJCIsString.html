Method IsString

Parameters
value - A supplied value.

Returns

true if a string; otherwise false.

Syntax

JavaScript
static IsString(value)

Checks if the value is a string.

Example

JavaScript
// Checks if the value is a string.
IsString()
{
  let number = "1";
  let result = LJC.IsString(number);

  // result:
  // true;

  number = 1;
  result = LJC.IsString(number);

  // result:
  // false;
}
