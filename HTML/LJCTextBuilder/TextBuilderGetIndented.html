Method GetIndented

Parameters
text - The add text.

Returns

The potentially indented new text value.

Syntax

JavaScript
GetIndented(text)

Gets a new potentially indented line.

Example

JavaScript
// Gets a new potentially indented line.
GetIndented()
{
  const tb = new LJCTextBuilder();
  tb.AddIndent();

  // Example Method:
  const result = tb.GetIndented("This is some indented text.");

  // result:
  // "  This is some indented text."
}
